## Overview

This portfolio is designed to impress technical recruiters and hiring managers for internship roles in Machine Learning Engineering, Data Engineering, and Data Science. It features a dark-fantasy aesthetic inspired by the Elden Ring universe, providing an epic and immersive experience.

## Technologies Used

- **Frontend Framework:** Next.js
- **Styling:** Tailwind CSS, SCSS
- **3D Graphics:** Three.js
- **Animation:** GSAP/Framer Motion
- **Data Fetching:** GitHub API, LinkedIn API

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/tarnished-of-data-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd tarnished-of-data-portfolio
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Create a `.env` file based on the `.env.example` file and add your API keys.
5. Start the development server:
   ```
   npm run dev
   ```

## API Integration

This portfolio automatically pulls content from GitHub and LinkedIn using their respective APIs. Ensure you have valid API keys and set them in your `.env` file.

## Customization

You can easily customize the site settings by modifying the `src/config/siteConfig.ts` file. Here, you can change your name, tagline, and social media links.
