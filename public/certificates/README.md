# Certificate Upload Instructions

Upload your PDF certificates to this folder with the following naming convention:

## Certificates:
1. **Fundamentals_of_Data_Analytics_Analyttica.pdf** - Fundamentals of Data Analytics from Analyttica
2. **Basics_of_Python_Infosys.pdf** - Basics of Python from Infosys
3. **Microsoft_Azure_Fundamentals.pdf** - Microsoft Azure Fundamentals from Microsoft

## Experience/Internship Certificates:
1. **AWS_Cloud_Virtual_Internship_EduSkills.pdf** - AWS Cloud Virtual Internship from EduSkills
2. **ML_Internship_SkillSizire.pdf** - Machine Learning Internship from SkillSizire
3. **AI_ML_Internship_IBM_SkillBuild.pdf** - AI & Machine Learning Internship from IBM SkillBuild / edunet

## Note:
- The "View Certificate" buttons on the website will automatically link to these PDF files
- Make sure the file names exactly match the names specified above
- Once you upload the PDFs, the links will work immediately
